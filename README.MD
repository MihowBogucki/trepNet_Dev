# TrepNet Application

The TrepNet app intergrates with **Firebase** and **Ionic Framework** to build a login app for users to send messages to each other

* User Authentication.
* Profile Page.

## Installation

Clone this repository using ssh:

After project has been cloned:

```sh
$ npm install
```

Then 

```sh
$ ionic serve
```
